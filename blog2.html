Notas sobre CSS

Aquí encontrarás algunas notas y ejemplos sobre CSS para mejorar el diseño de tu página web.

1. Reset y configuración global

Es común resetear los estilos predeterminados del navegador para evitar inconsistencias:

* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
    font-family: Arial, sans-serif;
}
Nota: box-sizing: border-box asegura que el padding y el borde estén incluidos en el ancho y alto de los elementos.

2. Animación de fondo

Puedes crear animaciones de fondo con gradientes y keyframes:

body {
    background: linear-gradient(90deg, rgba(92,7,1,1) 0%, rgba(153,36,79,1) 33%, rgba(226,36,0,1) 53%);
    background-size: 400% 400%;
    animation: gradient 15s ease infinite;
}

@keyframes gradient {
    0% { background-position: 0% 50%; }
    50% { background-position: 100% 50%; }
    100% { background-position: 0% 50%; }
}
Nota: Usa @keyframes para definir animaciones y animation para aplicarlas.

3. Header fijo con desenfoque

Un header fijo con efecto de desenfoque puede mejorar la estética de tu página:

header {
    position: fixed;
    top: 0;
    width: 100%;
    background-color: rgba(0, 0, 0, 0.8);
    backdrop-filter: blur(5px);
    display: flex;
    align-items: center;
    justify-content: space-around;
}
Nota: backdrop-filter: blur(5px) aplica un efecto de desenfoque al fondo del header.

4. Menús horizontales y verticales

Puedes crear menús desplegables con CSS:

.menu-horizontal {
    display: flex;
    list-style: none;
}

.menu-horizontal li:hover .menu-vertical {
    display: block;
}

.menu-vertical {
    display: none;
    position: absolute;
    background-color: rgba(0, 0, 0, 0.7);
    backdrop-filter: blur(5px);
}
Nota: Usa position: absolute para posicionar el menú vertical debajo del ítem del menú horizontal.

5. Secciones con sombras y bordes redondeados

Puedes mejorar el diseño de las secciones con sombras y bordes redondeados:

section {
    background-color: rgba(17, 17, 17, 0.9);
    border-radius: 10px;
    box-shadow: 0 4px 6px rgba(0, 0, 0, 0.1);
}
Nota: box-shadow agrega sombras, y border-radius redondea los bordes.

6. Diseño responsive

Usa media queries para adaptar tu diseño a diferentes dispositivos:

@media (max-width: 768px) {
    nav {
        flex-wrap: wrap;
        justify-content: center;
    }
    
    h1 {
        font-size: 2rem;
    }
}
Nota: Las media queries permiten ajustar estilos según el ancho de la pantalla.
