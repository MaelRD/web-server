Objetivos

      Impementar Hojas de estilo:

      Aplicar selectores CSS para estilizar la página web desarrollada en la Práctica 2, mejorando su apariencia visual y organización. Esto incluye el uso de propiedades como colores, fuentes, márgenes, paddings y bordes para lograr un diseño coherente y atractivo.

      Crear un Layout Responsivo:

      Diseñar un layout que sea adaptable a diferentes dispositivos (escritorio, tablet, móvil) utilizando media queries. Esto garantizará que la página se visualice correctamente en cualquier pantalla, mejorando la experiencia del usuario.

      Desarrollar Menús Multinivel:

      Implementar un menú horizontal con al menos cinco secciones, cada una con un menú vertical desplegable que contenga cinco entradas. Estos menús deben ser funcionales y estéticamente consistentes, permitiendo una navegación intuitiva.

      Utilizar Flexbox y CSS Grid Layout:

      Crear dos propuestas de grid utilizando Flexbox y otras dos utilizando CSS Grid Layout. Esto permitirá comparar ambas tecnologías y entender sus ventajas y desventajas en la creación de layouts complejos.

      Incorporar Elementos de Contenido Variados:

      Asegurar que cada entrada del menú vertical enlace a una página HTML diferente, donde se utilicen diversos elementos como tablas, imágenes, listas, formularios y contenido multimedia (videos, iframes). Esto enriquecerá el contenido del sitio y demostrará el manejo de múltiples etiquetas HTML.

      Optimizar la Estructura Semántica:

      Utilizar elementos semánticos de HTML5 (como "header, nav, main, section, article y footer) para mejorar la accesibilidad y el SEO del sitio. Esto ayudará a los motores de búsqueda a entender mejor la estructura del contenido.
