Propósito"

El propósito de esta práctica es fortalecer las habilidades en el desarrollo web mediante la aplicación de técnicas avanzadas de CSS y la creación de layouts responsivos. A través de la modificación y mejora de la página web desarrollada en la Práctica 2, se busca profundizar en el uso de CSS para estilizar páginas de manera profesional, aprender a crear layouts responsivos con media queries y unidades relativas, y explorar las tecnologías de Flexbox y CSS Grid para diseñar estructuras modernas y flexibles. Además, se pretende mejorar la organización del código utilizando elementos semánticos de HTML5 y fomentar buenas prácticas de desarrollo. Otro objetivo clave es desarrollar menús interactivos, incluyendo menús horizontales, verticales multinivel y un menú hamburguesa para dispositivos móviles, optimizando la navegación y la experiencia del usuario. Asimismo, se busca integrar contenido dinámico y variado mediante el uso de tablas, imágenes, listas, formularios y iframes, enriqueciendo el sitio web. Esta práctica también fomenta la creatividad e innovación en el diseño, preparando para enfrentar desafíos reales en el desarrollo web al simular un entorno de trabajo con requerimientos específicos y plazos definidos. En resumen, el propósito es consolidar conocimientos en HTML y CSS, explorar nuevas herramientas y técnicas, y entregar un sitio web funcional, responsivo y visualmente atractivo.
