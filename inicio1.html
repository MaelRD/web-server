Introducción

En el ámbito del desarrollo web, la creación de sitios que no solo sean funcionales, sino también visualmente atractivos y responsivos, es una habilidad esencial. La Práctica 3 tiene como objetivo principal mejorar la página web desarrollada en la Práctica 2, incorporando hojas de estilo CSS para aplicar diseños más sofisticados y estructurados. Este ejercicio nos permitirá explorar el uso de selectores CSS, la creación de layouts responsivos y la implementación de grids utilizando tanto Flexbox como CSS Grid Layout.

En este proyecto, se ha diseñado un blog personal que aborda temas de desarrollo web, intereses personales como la música y los videojuegos, y una sección de contacto. La estructura del sitio incluye un header con un menú horizontal multinivel, un footer, y secciones de contenido que utilizan diversos elementos HTML como tablas, imágenes, listas y formularios. Además, se ha prestado especial atención a la responsividad del sitio, asegurando que sea accesible y funcional en dispositivos móviles mediante el uso de media queries y un menú hamburguesa.

A lo largo de esta práctica, se han implementado dos propuestas de grid utilizando Flexbox y otras dos con CSS Grid Layout, lo que permite comparar y contrastar las ventajas de cada enfoque. El menú horizontal cuenta con cinco secciones principales, cada una de las cuales despliega un menú vertical con cinco entradas, todas enlazadas a diferentes páginas HTML. Estas páginas secundarias presentan contenido variado, desde biografías y discografías hasta horarios de clase y formularios de contacto.

En resumen, esta práctica no solo refuerza los conceptos básicos de HTML y CSS, sino que también introduce técnicas avanzadas de diseño web, preparándonos para crear sitios web modernos, eficientes y adaptables a las necesidades del usuario actual.
