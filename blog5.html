Notas sobre Diseño Web

Aquí encontrarás notas resumidas sobre diseño web, incluyendo layouts con HTML, CSS y Grid.

1. Diseño de Layouts con HTML

En HTML, los layouts se pueden estructurar utilizando elementos semánticos como <header>, <nav>, <section>, <article>, <footer>, etc.

<header>Cabecera</header>
    <nav>Menú de navegación</nav>
    <section>
        <article>Contenido principal</article>
        <aside>Barra lateral</aside>
    </section>
    <footer>Pie de página</footer>
Nota: Usar elementos semánticos mejora la accesibilidad y el SEO.

Referencia: W3Schools - HTML Layout

2. Diseño de Layouts con CSS

CSS permite crear layouts responsivos y personalizados. Algunas técnicas comunes incluyen:

  • Float: Para posicionar elementos a la izquierda o derecha.
  • Flexbox: Para crear diseños flexibles y alineados.
  • Grid: Para crear layouts complejos con filas y columnas.
.container {
        display: grid;
        grid-template-columns: 1fr 1fr 1fr;
        gap: 10px;
    }
Nota: Flexbox es ideal para diseños unidimensionales, mientras que Grid es mejor para diseños bidimensionales.

Referencia: W3Schools - CSS Website Layout

3. CSS Grid

CSS Grid es una poderosa herramienta para crear layouts complejos. Algunos conceptos clave incluyen:

  • Grid Container: El contenedor que define el grid.
  • Grid Items: Los elementos dentro del grid.
  • Grid Lines: Las líneas que dividen el grid.
  • Grid Tracks: Las filas y columnas del grid.
.grid-container {
        display: grid;
        grid-template-columns: repeat(3, 1fr);
        grid-template-rows: auto;
        gap: 15px;
    }
Nota: CSS Grid es compatible con la mayoría de los navegadores modernos.

Referencia: W3Schools - CSS Grid
