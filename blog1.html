Notas sobre HTML

Aquí encontrarás algunas notas básicas sobre HTML para ayudarte a crear páginas web.

1. Estructura básica de un documento HTML

Todo documento HTML debe tener una estructura básica:

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Título de la página</title>
</head>
<body>
    <!-- Contenido de la página -->
</body>
</html>
Nota: El <!DOCTYPE html> define que el documento es HTML5.

2. Etiquetas comunes

Algunas de las etiquetas más utilizadas en HTML son:

Encabezados

Los encabezados se definen con las etiquetas <h1> a <h6>:

<h1>Título principal</h1>
<h2>Subtítulo</h2>

Párrafos

Los párrafos se crean con la etiqueta <p>:

<p>Este es un párrafo de texto.</p>

Enlaces

Los enlaces se crean con la etiqueta <a>:

<a href="https://www.ejemplo.com">Enlace a Ejemplo</a>

3. Listas

Puedes crear listas ordenadas y no ordenadas:

Lista no ordenada

<ul>
    <li>Elemento 1</li>
    <li>Elemento 2</li>
</ul>

Lista ordenada

<ol>
    <li>Primer elemento</li>
    <li>Segundo elemento</li>
</ol>

4. Imágenes

Para insertar imágenes, usa la etiqueta <img>:

<img src="ruta/de/la/imagen.jpg" alt="Descripción de la imagen">
Nota: El atributo alt es importante para la accesibilidad y el SEO.

5. Formularios

Los formularios se crean con la etiqueta <form>:

<form action="/submit" method="post">
    <label for="nombre">Nombre:</label>
    <input type="text" id="nombre" name="nombre">
    <input type="submit" value="Enviar">
</form>
